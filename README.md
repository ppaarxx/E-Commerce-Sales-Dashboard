<h1>E-Commerce Sales Dashboard</h1>

<p>This Power BI dashboard was created to analyze and visualize e-commerce sales data. It includes various charts and visualizations to provide insights into sales trends, profitability, and more.</p>

<h2>Features</h2>
<ul>
  <li>Interactive dashboard with filters and slicers</li>
  <li>Complex parameters for deep data analysis</li>
  <li>Variety of visualizations including bar, pie, and donut charts</li>
</ul>

<h2>How to Use</h2>
<ol>
  <li>Download the <code>.pbix</code> file.</li>
  <li>Open it in Power BI Desktop to explore the dashboard and customize it to your needs.</li>
</ol>
